# 策略收益衰减：机制与对策

> 来源：「策略为什么必然会出现收益能力的衰减？」

## 核心结论

策略上线后收益衰减是**结构性必然**，非运气问题。三个机制同时作用并层层放大。

## 三个衰减机制

### 机制一：选择原罪（赢家偏差）

- 筛选过程本身含未来信息——你挑中的那个策略，回测表现必然高于真实期望
- 即使做了样本内外划分，"按外样本表现挑策略"本身就是一次未公开的回测
- 极端回测表现 = 真信号 + 噪声，上线后回归均值是统计期望

### 机制二：信息优势稀释

- **被动稀释**：研报/论文/人才跳槽 → 知识扩散，类似 idea 会被独立重新发现
- **主动稀释**：订单流留指纹，高频对手盘和做市商画像你的交易行为

### 机制三：交易本身改变市场

- 做市商根据你的足迹调整报价，逐步吃掉你利用的低效
- 对手盘会学习，挂反向流动性薅你
- 同类资金涌入，共同抹平价格偏离
- 越短周期、越规律 → 越快被消化

## 衰减速度对照表

| 维度 | 衰减快 | 衰减慢 |
|------|--------|--------|
| 持仓周期 | tick/分钟/日内 | 周线/月线/中长期 |
| 信号复杂度 | 单因子/论文复现 | 多因子复合/系统级 |
| 容量 | 小容量高换手 | 大容量低换手 |
| 信号来源 | 公开K线/财报 | 另类数据/自研事件库 |
| 执行可见度 | 高频规律明显 | 低频拆单不规律 |
| 生态 | 公开因子/同行密集 | 冷门标的/少人覆盖 |

**反直觉**：回测 Sharpe 高不代表实盘前景好。真正决定的是——所在生态有多拥挤、对手盘有多容易复制。

## 对策

1. **持续迭代**：研究当产线，老策略下线前新策略已在路上
2. **多策略组合**：不同 alpha 来源 × 不同周期 × 不同标的池
3. **容量管理**：alpha 浓时控规模，连续赚钱后别猛加仓
4. **区分坏了 vs 风格不对路**：建识别+动态分配机制，别一回撤就下线
5. **理解市场结构 > 找因子**：产出方法论而非单个因子
6. **核心转变**：从"找一个能赚钱的策略" → "构建持续产出新策略的研究体系"
